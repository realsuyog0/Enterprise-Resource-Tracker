# Enterprise-Resource-Tracker
A streamlined, web-based platform designed to simplify inventory control, order management, and business analytics. Built with Python, Streamlit, and SQLite, it offers an intuitive interface for tracking product lifecycles from arrival to final sale. 

# Key Features of this webapp:
- Track product names, codes, cost prices, and unit quantities in real-time.
- Efficiently process customer orders and track pending status.
- Record completed sales and automatically calculate profit margins.
- Visualize sales growth and performance using interactive charts.
- Built-in features for database backups and system restoration.

# Tech Stack includes:
Frontend: Streamlit
Database: SQLite
Data Processing: Pandas
Visualization: Plotly

# How to Run?
1. Clone the repository:

git clone https://github.com/yourusername/Enterprise-Resource-Tracker.git
cd Enterprise-Resource-Tracker

2. Install dependencies:

pip install -r requirements.txt

3. Launch the application:

streamlit run app.py
